## 🚀 **Quick Start Guide**

### 🛠️ **Prerequisites Checklist**
- ✅ **Flutter SDK** → Version 3.0.0 or higher
- ✅ **Development IDE** → Android Studio or VS Code with Flutter plugins
- ✅ **Android Development** → Android SDK and platform tools
- ✅ **iOS Development** → Xcode (macOS only) for iOS deployment

### ⚡ **Lightning-Fast Installation**

1. **📥 Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/flutter-calculator.git
   cd flutter-calculator
   ```

2. **📦 Install Dependencies**
   ```bash
   flutter pub get
   ```

3. **🎯 Launch the App**
   ```bash
   flutter run
   ```

4. **🎉 Enjoy Your Calculator!**

## 🔧 **Android Studio Development Setup**

### 🎯 **Step 1: Environment Preparation**
1. 📥 **Install Android Studio** → Download from [official website](https://developer.android.com/studio)
2. 🔌 **Add Flutter Plugin** → Install Flutter and Dart plugins in Android Studio
3. 📦 **Setup Flutter SDK** → Download from [Flutter.dev](https://flutter.dev/docs/get-started/install)
4. ⚙️ **Configure Path** → Set Flutter SDK path in Android Studio settings

### 📂 **Step 2: Project Import Magic**
1. 🚀 **Launch Android Studio** → Open the IDE
2. 📁 **Import Project** → Select "Open an existing Android Studio project"
3. 🎯 **Navigate to Folder** → Find your cloned repository location
4. ⏳ **Wait for Magic** → Let gradle sync and indexing complete

### 📱 **Step 3: Device Configuration**
1. 🔧 **Enable Developer Mode** → Tap build number 7 times on Android device
2. 🐛 **USB Debugging ON** → Enable in Developer Options
3. 🔌 **Connect Device** → Use USB cable to connect
4. 🤖 **Or Use Emulator** → Launch from AVD Manager for virtual testing

### ▶️ **Step 4: Launch Configuration**
1. 🎯 **Select Target** → Choose device from dropdown menu
2. ▶️ **Hit Play** → Click green play button or press Shift+F10
3. 🎉 **Enjoy** → Watch your app build and install automatically

## 🐛 **Troubleshooting Guide**

### ⚡ **Quick Fixes for Common Issues**
- 🧹 **Build Failures** → Run `flutter clean` followed by `flutter pub get`
- 🔌 **Device Not Found** → Verify USB debugging enabled and drivers installed
- 🐌 **Performance Issues** → Switch to Release mode for optimal performance
- 📱 **UI Problems** → Check device orientation and screen size compatibility
- 🔧 **Dependency Issues** → Update Flutter SDK and run `flutter doctor`
